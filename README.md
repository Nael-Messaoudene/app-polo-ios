#  Polo - Application iOS

Polo is an iOS application developed to create a surprised travel from A to Z. 

### Installation

Clone the project and install the pods dependencies

```sh
$ pod install
```

### Architecture

```
├── Pods
├── PodsTravel.xcodeproj
├── PodsTravel.xcworkspace
├── PoloTravel
│   ├── Extensions
│   ├── Protocols
│   ├── SupportingFiles 
│   ├── ViewModels
│   ├── ViewControllers
├── PoloTravelTests
├── Podfile
├── Podfile.lock

```
